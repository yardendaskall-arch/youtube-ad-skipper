# youtube-ad-skipper
this shows a gui of cool youtube features

# Steps
1. go to youtube
2. click the star and click done
3. click on 3 dots (on chrome) an find bookmarks and list and right-click youtube click edit
4. in the url bar paste in the code in script and in the name bar put a name for it
5. go back to bookmarks and lists and click show bookmarks bar
6. click on the button with the name you gave it
7. look at the interface and enjoy.
